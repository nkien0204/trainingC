Mot so bai tap ren luyen ky nang lap trinh C ¶

1. Cac thao tac tren chuoi: ¶

1.1. Viet chuong trinh kiem tra xem 1 chuoi chi chua cac chu so hay ko.

  • Vi du:
    • Neu chuoi la "abc1234" -> Tra ve gia tri false.
    • Neu chuoi la "6789989" -> Tra ve gia tri true.

1.2. Viet chuong trinh xoa het whitespace o ben trai va ben fai 1 chuoi.

  • Goi y: Cac ky tu whitespace bao gom: { ' ', '\f', '\n', '\r', '\t', '\v' }.

1.3. Viet chuong trinh chuan hoa 1 chuoi.

  • Goi y: 1 chuoi dc goi la da duoc chuan hoa khi:
    • Ko co whitespace o truoc va sau chuoi.
    • Giua 2 tu trong chuoi chi cach nhau boi 1 whitespace.
    • Moi tu chi co chu cai dung dau duoc viet hoa, cac chu cai khac viet thuong.
  • Vi du: Neu chuoi la " lIfe 'S gOOD! " thi chuoi sau khi da duoc chuan hoa se la "Life 's Good!".

1.4. Viet chuong trinh tim vi tri dau tien cua 1 chuoi con trong 1 chuoi dai.

  • Vi du: Neu chuoi dai la "I can fly", chuoi con la "can' thi fai in ra duoc vi tri dau tien la 2.

1.5. Viet chuong trinh tim tat ca cac vi tri cua 1 chuoi con trong 1 chuoi dai.

1.6. Viet chuong trinh tra ve chuoi dung sau 1 chuoi con trong 1 chuoi dai.

  • Vi du: Neu chuoi dai la "Love is blue", chuoi con la "is " thi chuoi tra ve se la "blue".

1.7. Viet chuong trinh tra ve chuoi dung truoc 1 chuoi con trong 1 chuoi dai.

  • Vi du: Neu chuoi dai la "Love is blue", chuoi con la "is " thi chuoi tra ve se la "Love "

1.8. Viet chuong trinh tra ve chuoi dung giua 2 chuoi con trong 1 chuoi dai.

  • Vi du: Neu chuoi dai la "Love is blue", chuoi con ben trai la "Love ", chuoi con ben phai la " blue" thi chuoi tra ve se la "is".

1.9. Viet chuong trinh noi cac chuoi voi nhau voi so chuoi khong xac dinh.

  • Vi du:
    • Neu dau vao la cac chuoi: { "I ", "am", " lucky" } thi dau ra se la chuoi "I am lucky".
    • Neu dau vao la cac chuoi: { "She", " is ", "so", " beautiful" } thi dau ra se la chuoi "She is so beautiful".
  • Goi y: Xay dung ham noi chuoi co giao dien nhu sau: 
    char *str_cat(char *str1, ...);

1.10. Viet chuong trinh convert 1 chuoi chua cac so nguyen (moi so cach nhau boi 1 dau cach) thanh 1 array cac so nguyen.

  • Vi du: Neu chuoi la "1 2 3" thi tra lai mang sau: { 1, 2, 3 }.
  • Goi y: Dung ham strtok.

1.11. Viet chuong trinh convert 1 chuoi bieu dien 1 so thap phan thanh 1 so co kieu float.

  • Vi du: Neu chuoi la "123.456" thi tra lai so 123.456

2. Cac thao tac tren file va he thong file: ¶

2.1. Viet chuong trinh doc va in ra cac so nguyen trong 1 text file theo thu tu, biet rang vi tri cac so nguyen trong file la ngau nhieu.

  • Vi du: Neu cau truc file la:

"1 4 5 7"
" 5 3 4 6"
thi phai in ra cac so nguyen 1, 4, 5, 7, 5, 3, 4, 6. Moi so cach nhau boi 1 dau cach.

2.2. Cung tuong tu nhu bai 2.1, nhung khi in ra phai dam bao cac so nguyen ko bi trung lap.

  • Vi du: Voi file o tren thi se in ra cac so nguyen 1, 4, 5, 7, 3, 6.

2.3. Tuong tu bai 2.1, nhung trong file ko chi chua cac so nguyen ma con chua ca cac so thap phan.

2.4. Viet chuong trinh ghi 10 object co type duoc dinh nghia duoi day vao 1 file co tren la save.bak 

   struct data {
     int type;
     double val;
     char buf[100];
   };

Sau do doc file save.bak va in ra gia tri cac field tuong ung voi tung object.

2.5. Viet chuong trinh kiem tra 1 file co ton tai ko.

2.6. Viet chuong trinh kiem tra 1 dir co ton tai ko.

2.7. Viet chuong trinh liet ke toan bo cac file va dir trong 1 thu muc.
